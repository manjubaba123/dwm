bananimu's dwm version
----------------------

Requirements
------------
- xlib headers
- make
- gcc
- terminus font
- (xclip for urxvt copy/paste)

Installation Instructions
-------------------------
    git clone https://github.com/manjubaba123/dwm.git
    cd dwm && sudo make clean install

Patches
-------
- gaplessgrid
- horizgrid
- push
- columns
